# Queue Docs

Place reference documents here that the AI agent should read before working on tasks.

These files are passed to Aider as **read-only context** via the `--read` flag.
The agent can see them but cannot edit them.

## Good candidates for this folder

- Architecture overviews
- API documentation
- Database schemas
- Style guides
- Business rules
- Anything the agent needs to "know" but not modify
